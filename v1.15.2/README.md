# v1.15.2

Исправления:
- исправлен список преподавателей: если сайт БелГУ не отдаёт полный список напрямую, приложение берёт преподавателей из сохранённого расписания твоей группы;
- при выборе такого преподавателя показываются его пары из сохранённого расписания группы;
- улучшены попытки загрузить список преподавателей с разных адресов сайта;
- исправлена пасхалка: старая встроенная картинка больше не показывается на несколько секунд перед нужной;
- теперь сначала показывается надпись «Загружаю пасхалку...», а потом сразу нужная картинка по ссылке;
- версия в настройках обновлена до `v1.15.2`.

Известные баги:
- если сайт БелГУ не отдаёт полный список преподавателей, приложение показывает преподавателей из кэша твоей группы, а не весь университетский список.
