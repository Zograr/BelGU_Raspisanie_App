# v1.19.7

Фикс проблемы, когда 2 / Н / НН не появлялись:
- повторно проверена v1.17.14;
- `buildCollectGradesScript` оставлен ровно из v1.17.14;
- добавлено ожидание полной загрузки таблицы журнала:
  - если предметы уже есть, а поле mark ещё пустое, приложение ждёт и пробует собрать снова;
  - это исправляет ситуацию, когда новая версия собирала таблицу слишком рано.
- автосбор после выбора недели теперь стартует позже;
- при подгрузке семестра каждая неделя также собирается с ожиданием.

Почему в v1.17.14 работало:
- там ты вручную нажимал сбор уже после загрузки таблицы;
- в новых версиях автосбор мог срабатывать раньше, чем сайт дорисовывал оценки.
