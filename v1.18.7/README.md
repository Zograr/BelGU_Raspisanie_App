# v1.18.7

Изменения в оценках:
- кнопка «Подгрузить семестр» перенесена на главное место вместо «Собрать выбранную неделю»;
- основное действие теперь собирает весь выбранный семестр;
- «Собрать неделю» оставлена как дополнительная маленькая кнопка;
- кнопка «Показать 2 / Н / НН» убрана из общих действий;
- теперь она появляется уже в прогруженном списке оценок, то есть после сбора недели или семестра.

Дизайн:
- кнопки быстрого доступа чуть изменены;
- стиль остался нейтральным и одинаковым;
- фон кнопок ближе к обычным карточкам;
- добавлена лёгкая тень, чтобы кнопки не сливались с фоном.

Сохранено из v1.18.6:
- фикс установки APK через приложение;
- правильный `ApkFileProvider`;
- проверка скачанного APK;
- галочка «Не выходить из системы»;
- автоподгрузка недели после выбора.

Версия:
- versionCode = 65
- versionName = 1.18.7

Известные баги:
- «Не выходить из системы» зависит от cookies сайта; если сайт завершит сессию, потребуется новый вход.
- парсер оценок зависит от HTML сайта dekanat.bsuedu.ru.
