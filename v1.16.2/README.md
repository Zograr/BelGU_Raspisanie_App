# v1.16.2

Изменения:
- в расписании преподавателя сверху добавлен блок из 4 кнопок как на макете:
  - большая кнопка «Обновить»;
  - «Предыдущая неделя»;
  - «Текущая неделя»;
  - «Следующая неделя»;
- кнопки недели нажимают соответствующие ссылки недели на странице БелГУ и заново собирают расписание;
- дни без пар теперь тоже показываются;
- если у преподавателя в день нет занятий, отображается:
  - день;
  - `0 пар`;
  - карточка `Пар нет`;
- сохранён дизайн недельного просмотра преподавателя.

Известные баги:
- переключение недели зависит от текста ссылок на сайте БелГУ. Если сайт изменит названия кнопок, понадобится хотфикс.
