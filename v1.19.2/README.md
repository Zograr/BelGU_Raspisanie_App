# v1.19.2

Фиксы оценок и посещаемости:
- исправлен поиск проблемных записей:
  - теперь кнопка «Показать 2 / Н / НН» проверяет не только поле оценки, но и посещаемость;
  - Н и НН корректно считаются пропусками;
  - 2 теперь не должна теряться при сборе таблицы.
- переработан парсер журнала:
  - П считается как «присутствовал»;
  - О считается как «онлайн»;
  - Н/НН считаются пропусками;
  - П/О больше не показываются как «тип».
- в карточках больше нет «тип: п» или «тип: о»;
- вместо этого показывается нормальный статус:
  - «присутствовал»;
  - «онлайн»;
  - «пропуск: Н»;
  - «пропуск: НН».

Сохранено:
- запоминание пароля через Android Keystore;
- анимации v1.19.1;
- исправление установки APK через приложение;
- подгрузка семестра;
- исправления режима «Проблем нет».

- versionName = 1.19.2
