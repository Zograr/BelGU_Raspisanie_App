# v1.17.10

Фиксы выбора семестра/недели:
- исправлен баг, когда при нажатии «Выбрать неделю» мог открываться список семестров;
- кнопка недели теперь ищет только список с датами недели;
- fallback на последний `select` убран, чтобы не подставлялся список семестров;
- если недели ещё не загрузились, приложение покажет понятное сообщение и попросит подождать.

Улучшения просмотра оценок:
- карточки по дням стали заметнее;
- убран лишний «Источник» из обычных карточек, чтобы список был чище;
- добавлена дата в деталях карточки;
- сводка теперь показывает:
  - всего записей;
  - сколько с оценками;
  - сколько двоек;
  - сколько Н/НН;
  - сколько без оценки.

Новая кнопка:
- добавлена кнопка «Показать проблемы: 2 / Н / НН»;
- по ней показываются только двойки и пропуски;
- есть кнопка возврата ко всем оценкам.

Сохранено:
- фиксы установки APK из v1.17.9;
- улучшенный интерфейс оценок из v1.17.8;
- проверка валидности APK.

Известные баги:
- функция оценок зависит от HTML таблицы dekanat.bsuedu.ru. Если сайт изменит таблицу, понадобится хотфикс.
