# v1.17.11

Хотфикс выбора недели:
- исправлен баг v1.17.10, когда недели вообще не показывались;
- прошлый фильтр был слишком строгим и искал только варианты с полными датами и тире;
- теперь список недели ищется мягче:
  - варианты с датами `дд.мм` или `дд.мм.гггг`;
  - варианты со словом «неделя»;
  - варианты с диапазоном дат;
- список семестров больше не подставляется вместо недель;
