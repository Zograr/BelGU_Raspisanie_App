Дизайн быстрого доступа:
- кнопки быстрого доступа стали слегка цветными;
- все кнопки остаются в одном стиле;
- цвет отличается только мягким оттенком:

  - Обновить — тёплый жёлтый оттенок;
- в тёмной теме используются приглушённые цвета, чтобы не резало глаза.
