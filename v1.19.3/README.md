# v1.19.3

Фиксы журнала:
- П / О / Н / НН теперь показываются отдельной компактной строкой:
  - слева короткий бейдж: П, О, Н или НН;
  - справа понятный статус: Присутствовал, Онлайн, Пропуск;
  - текст больше не должен ломаться и переноситься непонятно.
- Убраны длинные чипы посещаемости из строки с парой.
- Исправлен поиск 2 / Н / НН:
  - теперь приложение дополнительно проверяет `raw`-строку таблицы после названия предмета;
  - это нужно на случай, если сайт кладёт оценку/пропуск не в отдельную ячейку, а рядом с другими данными;
  - `2`, `Н`, `НН` теперь должны отображаться как оценка/пропуск и попадать в список проблем.

Сохранено:
- запоминание пароля через Android Keystore;
- анимации v1.19.1;
- исправление установки APK через приложение;
- подгрузка семестра.
