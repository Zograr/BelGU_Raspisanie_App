# BelGU Schedule App

Приложение показывает расписание на сегодня для группы 90002595.

Изменения в этой версии:
- выбор подгруппы вынесен в верхнюю зелёную шапку;
- выбор открывается по нажатию как меню;
- аудитория и корпус выделяются отдельными плашками;
- выбранная подгруппа сохраняется после закрытия приложения.

Сборка APK: Build -> Generate App Bundles or APKs -> Generate APKs.
