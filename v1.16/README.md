# v1.16

Изменения:
- изменён дизайн расписания преподавателя;
- расписание преподавателя теперь отображается более похоже на недельный режим:
  - отдельные блоки дней;
  - счётчик пар на день;
  - компактные карточки пар;
  - предмет, время, группа, аудитория/корпус;
- добавлена кнопка «Следующая неделя» в расписании преподавателя;
- кнопка пытается нажать «Следующая неделя» на странице БелГУ и собрать новое расписание;
- сохранена кнопка «Обновить с сайта»;
- сохранён сбор расписания преподавателя через скрытый WebView.

Известные баги:
- кнопка «Следующая неделя» зависит от разметки сайта БелГУ. Если сайт изменит текст/кнопку перехода недели, может понадобиться хотфикс.
