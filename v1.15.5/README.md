# v1.15.5

Изменения:
- пасхалка больше не загружает картинку из интернета;
- картинка пасхалки встроена прямо в приложение и появляется сразу;
- встроенная картинка заменена на прикреплённое изображение;
- список преподавателей больше не ограничивается только преподавателями текущей группы: приложение дополнительно пытается получить общий список из раздела расписания преподавателей;
- добавлена кнопка «Открыть по ссылке», чтобы можно было вставить прямую ссылку вида `teachers/index.php?teacher=ID`;
- расписание преподавателя теперь отображается карточками, а не длинным текстом;
- в карточках расписания преподавателя показываются:
  - тип занятия: лек., лаб., пр., зач., дифф.зач.;
  - время;
  - предмет;
  - группа;
  - аудитория и корпус;
- пары преподавателя из сохранённого расписания группы тоже отображаются в новом карточном виде.

Известные баги:
- полный общий список преподавателей зависит от того, отдаёт ли сайт БелГУ список в HTML. Если сайт не отдаёт общий список, можно открыть преподавателя по прямой ссылке.
