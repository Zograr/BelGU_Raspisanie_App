# v1.18.2

Изменения именно в основном меню настроек:
- сверху новая большая карточка «Текущие настройки»;
- добавлен раздел «Быстрый доступ»;
- быстрые кнопки: Оценки, Преподы, Темы, Обновить;
- разделы переименованы и разложены понятнее:
  - Расписание;
  - Учёба;
  - Оформление и приложение.

Фиксы в «Оценки и посещаемость»:
- «Открыть журнал» заменено на «Сбросить выбор»;
- сброс очищает семестр, неделю и старый список оценок;
- при выборе нового семестра старая неделя очищается;
- нельзя собрать оценки без выбранных семестра и недели.
