# v1.15.10

Хотфикс внутри этой же версии:
- исправлена ошибка сборки `cannot find symbol variable rawText`;
- `day.rawText` заменено на правильное поле `day.fullText`;
- номер версии НЕ повышался, потому что это не новая функция, а исправление ошибки сборки.

Изменения v1.15.10:
- расписание преподавателя собирается тем же способом, что и обычное расписание группы;
- используется скрытый WebView: приложение открывает страницу преподавателя, ждёт загрузку и собирает текст расписания со страницы;
- используется общий парсер дней и пар: даты, пары, время, предмет, группа, аудитория и корпус;
- кнопка обновления преподавателя называется «Обновить с сайта»;
- добавлена Аркаева Людмила Васильевна (`teacher=73019`) во встроенный список;
- если преподаватель есть во встроенном списке, приоритетно открывается его полное расписание с сайта;
- если расписание пустое, показывается понятное сообщение без мусорного текста страницы.

Известные баги:
- если сайт БелГУ не отдаст страницу преподавателя или неделя пустая, приложение покажет «Пары не найдены».
