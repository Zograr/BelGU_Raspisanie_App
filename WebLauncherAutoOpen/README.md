# WebLauncherAutoOpen

Простое Android-приложение: при нажатии на иконку приложения сразу открывает указанный сайт в браузере.
Ничего нажимать внутри приложения не нужно.

## Где поменять сайт

Открой файл:

`app/src/main/res/values/strings.xml`

И замени:

```xml
<string name="site_url">https://example.com</string>
```

на свой сайт, например:

```xml
<string name="site_url">https://zograr.ru</string>
```

## Где поменять название приложения

В этом же файле измени:

```xml
<string name="app_name">Мой сайт</string>
```

## Как собрать APK

Открой проект в Android Studio и нажми:

`Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`

После установки: нажимаешь на иконку приложения — сайт сразу открывается в браузере.

Если на телефоне нет браузера по умолчанию, Android может один раз показать выбор браузера. Это системное поведение.
