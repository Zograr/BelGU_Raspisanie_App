# v1.19.4

Фиксы журнала:
- исправлена ошибка сборки из-за неверного символа `\` в Java-строках;
- парсер оценок стал шире:
  - читает не только видимый текст ячеек;
  - дополнительно смотрит `value`, `title`, `aria-label`, `data-*`;
  - это важно, если сайт БелГУ прячет 2 / Н / НН в атрибутах или внутри элементов.
- если оценка не найдена в `mark`, приложение дополнительно ищет её в сырой строке после названия предмета.
- П / О / Н / НН показываются компактной строкой:
  - короткий бейдж;
  - понятный статус рядом;
  - без некрасивых переносов.

Версия:
- versionCode = 72
- versionName = 1.19.4

Известные баги:
- если сайт БелГУ отдаёт оценки через отдельный iframe или динамически подгружает таблицу после задержки, может потребоваться увеличить задержку сбора или добавить диагностику raw-строк.
