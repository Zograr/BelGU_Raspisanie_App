# v1.15.7

Изменения:
- исправлена логика версий: это следующий хотфикс после v1.15.6;
- поиск преподавателей переделан под сценарий как на сайте БелГУ;
- в разделе преподавателей теперь есть кнопка «Найти на сайте БелГУ»;
- вводишь фамилию или часть фамилии, нажимаешь кнопку — приложение пробует выполнить поиск через сайт;
- приложение пробует несколько вариантов поискового запроса, потому что сайт может использовать разные внутренние параметры формы;
- если сайт не отдаёт результат через форму, приложение дополнительно ищет совпадения по HTML расписания выбранной группы и сохранённому кэшу;
- результаты поиска открываются как расписание преподавателя;
- пасхалка осталась встроенной и появляется сразу;
- WebP пасхалки дополнительно сжат, чтобы приложение весило меньше.

Известные баги:
- точная внутренняя форма поиска БелГУ может отличаться от предположенной. Если сайт не отдаст результат по фамилии, приложение покажет найденных преподавателей из страницы группы и кэша.
