# v1.17.8

Улучшения интерфейса оценок:
- добавлен отдельный блок «Выбрано сейчас»;
- теперь всегда видно:
  - какой семестр выбран;
  - какая неделя выбрана;
- после выбора семестра/недели значения сразу обновляются на экране;
- кнопка «Собрать оценки» стала основной большой кнопкой;
- блок результата стал понятнее:
  - отдельно показывается выбранный семестр;
  - отдельно показывается выбранная неделя;
  - сводка по записям стала читабельнее;
- карточки оценок переработаны:
  - оценка стала крупнее;
  - предмет читается лучше;
  - номер пары/тип/пропуск вынесены в плашки;
  - тема и источник вынесены в отдельные строки.

Сохранено:
- проверка валидности APK из v1.17.7;
- улучшенный ApkFileProvider;
- исправления оценок из v1.17.6;
- скрытый WebView для журнала СПО.

Известные баги:
- функция оценок всё ещё зависит от HTML таблицы dekanat.bsuedu.ru. После теста может потребоваться ещё один хотфикс.
