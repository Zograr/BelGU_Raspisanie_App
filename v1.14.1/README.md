# v1.14.1

Изменения:
- меню выбора темы вынесено в отдельное окно, которое открывается как настройки;
- в окне тем есть переключатель «Светлые / Тёмные»;
- при выборе «Светлые» показываются только светлые темы;
- при выборе «Тёмные» показываются только тёмные темы;
- исправлен баг, из-за которого некоторые фоновые элементы оставались зелёными независимо от выбранной темы;
- выбранная тема по-прежнему сохраняется после перезапуска;
- версия в настройках обновлена до `v1.14.1`.

Известные баги:
- на момент сборки известных багов нет.
