# v1.19.9

Фикс режима «Подгрузить семестр»:
- обычный выбор «семестр → неделя» уже отображал оценки нормально;
- проблема была именно в автоматическом проходе по всем неделям;
- после выбора каждой недели приложение теперь ждёт дольше;
- если таблица недели ещё не дорисовала оценки, приложение повторяет сбор до 7 раз;
- итоговый список за семестр дополнительно перерисовывается, чтобы его не затирали служебные сообщения;
- проверка готовности недели теперь использует тот же fallback-поиск, что и обычный просмотр недели.
