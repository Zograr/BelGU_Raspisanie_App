# v1.19.16

Улучшения после v1.19.15.

Версия:
- versionCode = 84
- versionName = 1.19.16

Что изменено:
- «Тип» теперь показывается значками с буквами:
  - П — присутствовал;
  - О — онлайн;
  - Н — пропуск;
  - НН — двойной пропуск.
- блок «Выбрано сейчас» получил новый дизайн:
  - отдельные вертикальные плашки;
  - длинная неделя не должна съезжать.
- у больших подсказок добавлена маленькая кнопка «Скрыть подсказку»:
  - подсказка и кнопка скрываются с анимацией;
  - после скрытия они больше не появляются.
- исправлено поведение при истёкшей сессии:
  - вместо тупика «Нужен вход» приложение возвращает форму входа;
  - показывает сообщение, что нужно войти заново;
  - можно сразу нажать «Войти сохранённым паролем».

Известные баги:
- критических известных багов нет.
