# v1.13

Изменения:
- добавлен виджет рабочего стола с ближайшей парой;
- виджет показывает статус, время, предмет, аудиторию/корпус и время последнего обновления;
- нажатие на виджет открывает приложение;
- добавлено автообновление расписания каждый день в 7:00;
- автообновление обновляет сохранённое расписание и сам виджет;
- после перезагрузки телефона автообновление планируется заново.

Важно:
- Android может задерживать будильник автообновления из-за энергосбережения.
- Для более стабильной работы можно отключить жёсткую экономию батареи для приложения.

Известные баги:
- автообновление через фон работает проще, чем обновление из приложения, поэтому если сайт БелГУ сильно изменит HTML, обновление из приложения может быть точнее.
