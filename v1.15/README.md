# v1.15

Изменения:
- добавлено расписание преподавателей;
- в настройках появилась кнопка «Расписание преподавателей»;
- список преподавателей загружается с сайта БелГУ и сортируется по алфавиту;
- сверху есть поиск по преподавателям;
- при выборе преподавателя открывается его расписание;
- добавлен выбор времени автозагрузки расписания на сегодня;
- автозагрузка больше не привязана только к 7:00;
- кнопка «Назад» телефона теперь работает как нормальная навигация:
  - из выбора тем возвращает в настройки;
  - из расписания преподавателя возвращает в список преподавателей;
  - из списка преподавателей возвращает в настройки/расписание;
  - из настроек возвращает к расписанию;
- добавлена секретная пасхалка: много раз нажми на версию приложения;
- пасхалка закрывается кнопкой «Назад».

Известные баги:
- список преподавателей зависит от HTML сайта БелГУ. Если сайт изменит форму, список может перестать загружаться.
