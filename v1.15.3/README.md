# v1.15.3

Исправления:
- переделана загрузка списка преподавателей;
- теперь приложение в первую очередь берёт преподавателей прямо со страницы расписания выбранной группы;
- приложение ищет ссылки вида `teachers/index.php?teacher=ID`, которые есть у преподавателей на сайте БелГУ;
- если ссылка преподавателя находится рядом с иконкой, приложение пытается найти ФИО рядом с этой ссылкой;
- если сайт не отдаёт ссылки, остаётся запасной вариант: преподаватели из сохранённого расписания группы;
- список по-прежнему сортируется по алфавиту;
- при выборе преподавателя с найденным ID открывается именно расписание преподавателя с сайта.

Известные баги:
- если сайт БелГУ поменяет HTML ссылок преподавателей, парсер снова может потребовать правку.
