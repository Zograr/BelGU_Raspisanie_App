# v1.17.14

Что исправлено:
- убран неудачный отдельный сбор проблем по всем неделям, который показывал не то;
- кнопка «Показать 2 / Н / НН» теперь работает от уже собранного списка оценок;
- логика такая:
  1. нажимаешь «Собрать оценки»;
  2. приложение показывает нормальный список;
  3. нажимаешь «Показать 2 / Н / НН»;
  4. приложение фильтрует именно эти уже собранные данные.

Почему так:
- этот способ уже показывал правильные пропуски и двойки;
- больше нет отдельного кривого парсинга, который пытался угадывать данные заново.

Дизайн:
- кнопки перенесены в отдельный блок «Действия»;
- главная кнопка «Собрать оценки» крупная сверху;
- «Семестр» и «Неделя» стоят рядом;
- «Показать 2 / Н / НН» и «Открыть журнал» стоят рядом;
- «Войти заново» вынесена ниже отдельно;
- результат проблем подписан как «из собранных оценок».

Сохранено:
- фиксы установки APK из v1.17.9;
- улучшенный интерфейс оценок;
- обычный просмотр оценок по выбранной неделе.

Известные баги:
- чтобы открыть «2 / Н / НН», сначала нужно нажать «Собрать оценки». Если нужны проблемы по всему семестру, позже сделаем это только после того, как точно разберём HTML таблицы сайта.
