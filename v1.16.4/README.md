# v1.16.4

Хотфикс сборки:
- добавлен `gradle.properties`;
- включено `android.useAndroidX=true`, потому что FileProvider берётся из AndroidX Core;
- исправлена ошибка `:app:checkDebugAarMetadata`, связанная с AndroidX-зависимостью.

Изменения v1.16.4:
- исправлена ситуация, когда загрузка зависала на «Ожидаю начала загрузки»;
- приложение больше не полагается только на системный DownloadManager;
- APK теперь скачивается напрямую внутри приложения через HttpURLConnection;
- прогресс обновляется сразу:
  - сколько скачано;
  - общий размер, если GitHub его отдаёт;
  - заполняющаяся шкала;
- после скачивания значок меняется на файл;
- по нажатию на файл открывается стандартное Android-окно установки/обновления;
- если APK уже скачан, приложение не качает его повторно, а сразу показывает файл для установки;
- добавлен FileProvider для безопасного открытия APK на Android 7+.

Известные баги:
- если GitHub не отдаёт общий размер файла, будет показан индикатор без точного процента, но количество скачанных байт будет обновляться.
