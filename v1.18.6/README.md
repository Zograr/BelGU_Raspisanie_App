# v1.18.6

Главный фикс обновлений из приложения:
- исправлена причина ошибки «Возникла проблема с файлом приложения»;
- `MainActivity` сохранял путь к скачанному APK в `schedule_cache`;
- `ApkFileProvider` раньше искал этот путь в `schedule_prefs`;
- из-за этого системный установщик Android получал content:// ссылку, но не мог открыть сам APK;
- через ADB установка работала, потому что ADB ставит APK напрямую и не использует `ApkFileProvider`.

Дополнительные фиксы обновления:
- перед открытием установщика приложение проверяет, что скачанный файл похож на APK/ZIP;
- если GitHub отдаст не APK, а HTML/ошибку/не тот asset, приложение покажет понятную ошибку;
- старый битый APK из кэша больше не переиспользуется;
- для установки сначала используется `ACTION_INSTALL_PACKAGE`, потом fallback через `ACTION_VIEW`;
- исправлен User-Agent версии.

Сохранено из v1.18.5:
- нейтральные одинаковые кнопки быстрого доступа;
- галочка «Не выходить из системы»;
- автоподгрузка оценок после выбора недели;
- кнопка «Подгрузить семестр».

Версия:
- versionCode = 64
- versionName = 1.18.6

Известные баги:
- «Не выходить из системы» зависит от cookies сайта; если сайт завершит сессию, потребуется новый вход.
- парсер оценок зависит от HTML сайта dekanat.bsuedu.ru.
