# v1.19.6

Главный фикс оценок:
- проверил v1.17.14 и перенёс из неё рабочий `buildCollectGradesScript`;
- новая версия снова использует старую рабочую логику поиска оценок:
  - 2;
  - Н;
  - НН;
  - 3/4/5;
  - зач/незач.
- убран широкий поиск по `class/id/name`, из-за которого появлялись ложные двойки;
- фильтр «Показать проблемы: 2 / Н / НН» снова работает по полю `mark`, как в v1.17.14.

Сохранён новый интерфейс:
- карточки оценок из новых версий;
- маленькие бейджи П / О / Н / НН без большой надписи;
- кнопка «Подгрузить семестр»;
- запоминание пароля через Android Keystore;
- исправление установки APK через приложение.

Анимации:
- переливание кнопок быстрого доступа сделано заметнее;
- фон кнопок постоянно мягко двигается градиентом;
- анимация подстраивается под выбранную тему.
