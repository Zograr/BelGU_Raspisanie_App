# BelGU Schedule App

Приложение показывает расписание на сегодня для выбранной группы БелГУ.

Что есть:
- выбор группы по нажатию на плашку «Группа ...» вверху;
- выбор 1/2 подгруппы или всех подгрупп в шапке;
- светлая и тёмная тема с плавным переключением;
- последнее расписание сохраняется в кэш;
- аудитория и корпус выделены отдельными контрастными блоками.

Сборка APK: Build -> Generate App Bundles or APKs -> Generate APKs.

Версия 1.7: цвета блоков аудитории и корпуса подогнаны под зелёную тему, из окна выбора группы убран пример номера.
