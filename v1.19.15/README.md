# v1.19.15

Фиксы и улучшения после стабильной v1.19.14.

Версия:
- versionCode = 83
- versionName = 1.19.15

Что изменено:
- кнопка «Открыть журнал» теперь переключается:
  - при открытии превращается в «Закрыть журнал»;
  - при закрытии скрывает WebView обратно;
  - переключение сделано с короткой анимацией.
- блок «Выбрано сейчас» переделан:
  - семестр и неделя теперь идут вертикально;
  - длинная неделя больше не должна съезжать.
- добавлена кнопка «Сбросить выбор»:
  - появляется только когда выбран семестр или неделя;
  - появляется/исчезает с анимацией.
- отображение типа занятия стало понятнее:
  - П → Присутствовал;
  - О → Онлайн;
  - Н → Пропуск;
  - НН → Двойной пропуск.
- вместо «тип: о/п» теперь показывается «Формат» и «Посещение».

Известные баги:
- критических известных багов нет.
