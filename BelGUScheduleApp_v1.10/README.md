## GitHub Releases / обновления

В `app/src/main/res/values/strings.xml` уже указано:

```xml
<string name="github_owner">Zograr</string>
<string name="github_repo">BelGU_Raspisanie_App</string>
```

Чтобы обновления работали:
1. Собери APK.
2. Создай GitHub Release с тегом выше версии приложения, например `v1.10`.
3. Прикрепи APK к Release.
4. В приложении открой ⚙ Настройки → Проверить обновления приложения.
