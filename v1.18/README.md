# v1.18

Изменения:
- переделан сбор оценок из журнала СПО;
- исправлена главная ошибка парсера: номер пары больше не считается оценкой;
- оценки `2`, `3`, `4`, `5`, `Н`, `НН` ищутся только после предмета и не берутся из номера пары;
- результат больше не зависит от маленького окна сайта;
- маленькое окно WebView скрыто, сайт работает внутри приложения только для загрузки данных;
- оценки показываются в большом кастомном интерфейсе:
  - день и дата;
  - предмет;
  - оценка;
  - номер пары;
  - тип занятия;
  - пропуск Н/НН;
  - тема занятия;
- добавлена сводка: всего записей, сколько с оценками, сколько Н/НН, сколько без оценки.

Сохранено:
- автообновление APK;
- установка APK через свой ApkFileProvider;
- фиксы журнала СПО и HTTPS/HTTP;
- расписание преподавателей.

Известные баги:
- функция оценок всё ещё зависит от HTML таблицы на dekanat.bsuedu.ru. Если у сайта будут другие колонки, парсер может потребовать ещё один хотфикс.
