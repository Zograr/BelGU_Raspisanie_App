## v1.9 — Первая версия приложения

### О приложении
Это приложение создано для удобного просмотра расписания занятий.  
Оно помогает быстро узнать пары на сегодня, посмотреть расписание на неделю и выбрать нужную подгруппу.  
Основная цель приложения — сделать просмотр расписания проще, быстрее и удобнее, чем через обычный сайт.

### Добавлено
- Добавлен просмотр расписания на сегодняшний день.
- Добавлена возможность выбора подгруппы.
- Добавлена поддержка светлой и тёмной темы.
- Добавлено отображение основной информации о парах:
  - название предмета;
  - время занятия;
  - аудитория;
  - преподаватель;
  - тип занятия, если он указан.

### Интерфейс
- Добавлен простой и понятный главный экран.
- Расписание отображается в виде карточек.
- Добавлены кнопки для быстрого просмотра расписания.
- Сделан базовый дизайн приложения для удобного использования.

### Известные баги
- Так как это первая версия приложения, возможны мелкие ошибки в отображении расписания.
- Возможны проблемы с отображением некоторых элементов в тёмной теме.
- Просмотр расписания на неделю ещё может быть не полностью удобным.
- Обновление данных пока может работать неидеально.

### Статус версии
- Версия рабочая.
- Приложение можно использовать для просмотра расписания.
- В следующих версиях планируется улучшить дизайн, настройки и работу с расписанием на неделю.
