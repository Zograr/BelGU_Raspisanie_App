# v1.17.6

Статус:
- это хотфикс внутри ветки v1.17;
- дальше исправления делаем как v1.17.7, v1.17.8 и так далее;
- новую крупную версию делаем только когда текущая функция станет стабильной.

Фиксы оценок:
- номер пары больше не считается оценкой;
- оценки `2`, `3`, `4`, `5`, `Н`, `НН` ищутся только после предмета;
- маленькое окно сайта скрыто, WebView работает внутри приложения только для загрузки данных;
- просмотр оценок сделан крупным кастомным интерфейсом:
  - день и дата;
  - предмет;
  - оценка;
  - номер пары;
  - тип занятия;
  - пропуск `Н/НН`;
  - тема занятия;
- добавлена сводка:
  - всего записей;
  - сколько с оценками;
  - сколько `Н/НН`;
  - сколько без оценки.

Сохранено:
- автообновление APK;
- установка APK через свой ApkFileProvider;
- фиксы журнала СПО и HTTPS/HTTP;
- расписание преподавателей.

Известные баги:
- функция оценок всё ещё зависит от HTML таблицы на dekanat.bsuedu.ru. После теста может потребоваться ещё один хотфикс.
