# v1.17.1

Хотфикс обновления:
- исправлена ошибка `Не удалось создать папку updates`;
- загрузчик APK теперь пробует несколько мест:
  - app external files / Download / updates;
  - внутреннюю папку приложения / updates;
  - cache / updates;
- если первая папка недоступна на прошивке, загрузка больше не должна сразу падать;
- сохранена функция оценок из v1.17.

Важно:
- если у тебя стоит v1.16.4 и именно она не может создать папку updates, эту версию, скорее всего, нужно поставить вручную один раз.
- после установки v1.17.1 следующие обновления должны скачиваться стабильнее.

Известные баги:
- функция оценок первая тестовая, парсер журнала может потребовать доработки после проверки на реальной таблице.
