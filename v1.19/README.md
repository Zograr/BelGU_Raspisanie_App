# v1.19

Это версия v1.19, потому что изменения уже крупнее обычного хотфикса.

Главное:
- добавлена мягкая анимация меню;
- добавлена анимация появления кнопок быстрого доступа;
- быстрые кнопки подстраиваются под выбранную тему;
- добавлена карточка «Сессия личного кабинета»;
- добавлена кнопка «Продлить сессию»;
- добавлены подсказки для системного автозаполнения Android;
- сохранены исправления оценок, подгрузки семестра и установки APK через приложение.
