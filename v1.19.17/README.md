# v1.19.17

Фиксы после v1.19.16.

Версия:
- versionCode = 85
- versionName = 1.19.17

Что изменено:
- кнопка «Скрыть подсказку» теперь скрывает только текст подсказки и саму кнопку;
- основная плашка остаётся на месте, чтобы экран не прыгал и не пропадал заголовок;
- исправлена обрезка оценок по датам:
  - раньше сбор строк журнала был ограничен `out.slice(0,300)`;
  - из-за этого оценки могли доходить только до 14.05.2026 во 2 семестре и до 25.12.2025 в 1 семестре;
  - теперь лимит 300 строк убран, приложение возвращает все найденные строки.
- остаточные подписи `тип:` заменены на `Формат:`.

Известные баги:
- если сам сайт БелГУ не отдаёт будущие/нужные строки в таблице, приложение не сможет их показать без данных с сайта.
