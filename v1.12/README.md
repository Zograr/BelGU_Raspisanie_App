# v1.12

Изменения:
- добавлено выделение текущей пары;
- в режиме «Сегодня» текущая пара выделяется цветом и меткой «сейчас»;
- в режиме «Неделя» текущая пара выделяется только у сегодняшнего дня;
- если APK новой версии уже скачан, повторное нажатие «Скачать» не качает файл заново, а сразу открывает установщик;
- если APK ещё скачивается, приложение сообщает об этом и не запускает повторную загрузку;
- сохранение ID скачанного APK работает даже после перезапуска приложения.

Известные баги:
- на момент сборки известных багов нет.
