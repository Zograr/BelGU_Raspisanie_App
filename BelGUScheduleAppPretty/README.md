# Расписание БелГУ — красивая версия

Приложение показывает расписание на сегодня для группы 90002595.

Что изменено:
- более аккуратная шапка;
- красивые карточки пар;
- отдельное выделение времени, типа занятия, преподавателя и аудитории;
- сохранение последней загруженной версии.

## Как собрать APK

1. Открой проект в Android Studio.
2. Дождись Gradle Sync.
3. Нажми `Build -> Generate App Bundles or APKs -> Generate APKs`.

Либо через Terminal:

```bat
.\gradlew.bat assembleDebug
```

Готовый APK будет здесь:

```text
app\build\outputs\apk\debug\app-debug.apk
```
