# v1.19.5

Фиксы оценок:
- убран слишком широкий поиск, из-за которого двойки появлялись там, где их не должно быть;
- оценки теперь ищутся только в ячейках после названия предмета;
- class/id/name больше не используются как источник оценки, потому что из-за них могли появляться ложные «2»;
- обычные оценки 3/4/5 не должны ломаться.

Анимации:
- добавлено постоянное мягкое переливание кнопок быстрого доступа;
- переливание подстраивается под выбранную тему;
- оставлены анимации нажатия и появления.

Версия:
- versionCode = 73
- versionName = 1.19.5

Известные баги:
- если БелГУ прячет оценку только в class/id, она может не находиться. Для точного исправления нужна диагностика raw/cells строки.
