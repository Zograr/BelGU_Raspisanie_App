# v1.18.5

Изменения быстрого доступа:
- кнопки быстрого доступа сделаны нейтральными и одинаковыми;
- убрана полупрозрачная надпись сверху;
- быстрые кнопки теперь визуально однотипные.

Оценки и посещаемость:
- добавлена галочка «Не выходить из системы»;
- пароль не сохраняется в приложении;
- сохраняется только сессия сайта через cookies WebView;
- если сайт сбросит сессию, приложение попросит войти заново;
- при выборе недели оценки подгружаются автоматически;
- добавлена кнопка «Подгрузить семестр»;
- она проходит по неделям выбранного семестра и собирает оценки за весь семестр;
- в карточках оценок за весь семестр показывается неделя.

Фиксы:
- исправлен `setMinHeight` -> `setMinimumHeight`;
- сохранена UTF-8 кодировка;
- убраны возможные дубли методов;
- не используются AndroidX-зависимости.

Версия:
- versionCode = 63
- versionName = 1.18.5

Известные баги:
- если сайт dekanat.bsuedu.ru поменяет HTML таблицы, парсер оценок может потребовать хотфикс.
- «Не выходить из системы» зависит от cookies сайта; если сам сайт завершит сессию, потребуется новый вход.
