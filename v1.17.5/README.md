# v1.17.5

Хотфикс открытия установщика APK:
- исправлена проблема на Android/MIUI/профилях, когда после скачивания APK появлялось окно «Открыть в приложении» и текст «Не поддерживается личными приложениями»;
- убран `Intent.createChooser(...)`;
- APK теперь открывается напрямую через `Intent.ACTION_VIEW` с MIME `application/vnd.android.package-archive`;
- если первый способ не сработал, используется запасной `Intent.ACTION_INSTALL_PACKAGE`;
- перед запуском установщика приложение выдаёт `content://`-доступ всем подходящим установщикам через `grantUriPermission`.

Сохранено:
- свой `ApkFileProvider` без AndroidX;
- фикс журнала СПО из v1.17.4;
- раздел «Оценки и посещаемость».

Важно:
- если Android спросит разрешение «Установка неизвестных приложений», разреши его для приложения расписания.
- если системный установщик всё равно не появится, APK придётся поставить вручную из GitHub Release.
