# v1.16.1

Хотфикс обновлений:
- исправлена проблема, когда проверка обновлений могла падать с `HTTP 403` от GitHub API;
- добавлен запасной способ проверки через страницу GitHub Releases;
- если GitHub API отдаёт 403, приложение пытается определить последний релиз через `/releases/latest`;
- если API не отдаёт список assets, APK скачивается по прямому пути `releases/latest/download/app-debug.apk`;
- добавлены более нормальные HTTP-заголовки для GitHub: User-Agent, Cache-Control, X-GitHub-Api-Version;
- сохранены изменения v1.16: новый дизайн расписания преподавателя и кнопка «Следующая неделя».

Важно:
- если сейчас у тебя стоит v1.15.10 и именно она выдаёт HTTP 403, она сама себя уже не исправит.
- нужно один раз установить этот APK вручную, а следующие обновления должны проверяться лучше.

Известные баги:
- если в GitHub Release APK называется не `app-debug.apk`, запасной способ скачивания может не найти файл. Лучше прикреплять APK именно с таким названием.
