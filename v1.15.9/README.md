# v1.15.9

Исправления:
- добавлена кнопка «Обновить расписание преподавателя» внутри экрана преподавателя;
- при нажатии кнопки приложение заново загружает полное расписание преподавателя с сайта БелГУ;
- если преподаватель был найден как запись из расписания группы, приложение сначала пытается сопоставить его со встроенным списком преподавателей и открыть полное расписание с сайта;
- если на сайте у преподавателя на выбранной неделе нет пар, приложение больше не показывает длинный мусорный текст страницы;
- вместо этого показывается понятное сообщение «Пары не найдены»;
- сохранён встроенный список 70 преподавателей из v1.15.8;
- сохранена встроенная WebP-пасхалка.

Известные баги:
- если у преподавателя на сайте нет пар на выбранной неделе, приложение не может показать пары за другую неделю без отдельной кнопки выбора недели преподавателя.
