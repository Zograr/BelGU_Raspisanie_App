# v1.17.7

Хотфикс обновления APK:
- исправлена ситуация, когда APK из браузера устанавливается, а из приложения установщик пишет «Возникла проблема с файлом приложения»;
- добавлена проверка скачанного APK:
  - файл должен быть ZIP/APK;
  - внутри должен быть `AndroidManifest.xml`;
- если файл повреждён или GitHub отдал не APK, приложение удаляет его и не предлагает установку;
- старый повреждённый APK из памяти приложения теперь не используется повторно;
- улучшен `ApkFileProvider`:
  - отдаёт реальный размер файла;
  - поддерживает `openTypedAssetFile`;
- сохранены фиксы оценок из v1.17.6.

Важно:
- если ты уже стоишь на v1.17.5, этот хотфикс лучше поставить вручную один раз, потому что проблема находится именно в механизме открытия APK старой версии.
- после установки v1.17.7 следующие обновления должны проверять целостность файла перед установкой.

Известные баги:
- функция оценок всё ещё тестовая и может требовать доработки после проверки реальной таблицы.
