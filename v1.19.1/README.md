# v1.19.1

Сохранение пароля:
- добавлена галочка «Запомнить пароль на этом устройстве»;
- пароль не хранится открытым текстом;
- используется Android Keystore + AES/GCM;
- при следующем входе пароль подставляется автоматически;
- это помогает, если сайт БелГУ выкидывает из аккаунта каждые ~12 часов.

Важно:
- хранить пароль на устройстве безопаснее через Android Keystore, но это всё равно чувствительные данные;
- если телефон скомпрометирован/root-доступ у вредного ПО, абсолютной защиты нет.

Анимации:
- добавлена анимация нажатия быстрых кнопок;
- добавлена лёгкая анимация появления карточек;
- сохранена анимация карточки «Текущие настройки»;
- быстрые кнопки продолжают подстраиваться под выбранную тему.

Сохранено:
- карточка «Сессия личного кабинета»;
- кнопка «Продлить сессию»;
- автозаполнение Android;
- исправления оценок;
- фикс установки APK через приложение.
