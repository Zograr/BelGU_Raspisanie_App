# v1.17.13

Хотфикс сбора двоек и пропусков:
- отдельный сбор «Все 2 / Н / НН» теперь использует отдельный более мягкий парсер;
- парсер больше не зависит только от обычного формата строки с номером пары;
- теперь он ищет Н, НН и 2 по строкам таблицы и пытается подтянуть дату, предмет и тему;
- добавлена поддержка случаев, когда в строке есть дата и НН, но предмет находится выше по таблице;
- это должно лучше находить пропуски вроде `19.01.2026 НН`.

Улучшения дизайна:
- карточки проблем теперь показывают крупную оценку/пропуск слева;
- рядом показывается предмет;
- ниже показывается дата и день;
- список стал понятнее: когда, какой предмет и что стоит.

Сохранено:
- обычный просмотр оценок по неделе;
- сбор проблем по всем неделям семестра;
- фиксы установки APK.

Известные баги:
- если сайт хранит предмет и НН в разных нестандартных блоках, может понадобиться ещё один хотфикс после скрина/видео именно этой строки.
