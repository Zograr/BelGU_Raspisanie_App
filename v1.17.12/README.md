# v1.17.12

Что изменено:
- отдельная кнопка «Собрать все 2 / Н / НН» теперь собирает проблемы не только по текущей неделе;
- приложение проходит по всем неделям выбранного семестра;
- показывает отдельный список проблем:
  - неделя;
  - дата;
  - предмет;
  - оценка или Н/НН;
  - номер пары;
  - тип занятия;
  - тема, если сайт её отдаёт;
- добавлена кнопка «Пересобрать проблемы».

Важно:
- сначала выбери семестр;
- потом нажми «Собрать все 2 / Н / НН»;
- сбор может занять время, потому что приложение последовательно переключает недели внутри сайта.

Сохранено:
- обычный просмотр оценок по выбранной неделе;
- улучшенный интерфейс из v1.17.10/v1.17.11;
- фиксы установки APK.

Известные баги:
- если сайт dekanat.bsuedu.ru не подгружает недели через обычный select, сбор по всем неделям может требовать ещё одного хотфикса.
