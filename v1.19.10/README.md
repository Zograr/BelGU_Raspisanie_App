# v1.19.10

Сделано из чистой v1.19.9 без случайных имён методов.

Фиксы:
- исправлена подгрузка семестра:
  - после выбора семестра приложение ждёт обновления списка недель;
  - после выбора каждой недели ждёт 5.5 секунд;
  - если оценки ещё не появились, повторяет сбор до 10 раз.
- П / О больше не считаются оценками;
- Н / НН считаются пропусками;
- счётчики показывают только реальные оценки/пропуски;
- сохранены fallback-поиск проблем по raw/cells и диагностика строк журнала.

- versionCode = 78
- versionName = 1.19.10

Известные баги:
