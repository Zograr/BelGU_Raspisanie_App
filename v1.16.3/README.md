# v1.16.3

Изменения:
- добавлен экран прогресса скачивания обновления внутри приложения;
- после нажатия «Скачать» появляется окно:
  - значок загрузки;
  - полоса прогресса;
  - сколько скачано;
  - общий размер файла, если Android его отдаёт;
- если APK уже скачивается, повторное нажатие не запускает вторую загрузку, а показывает текущий прогресс;
- когда APK скачан, значок загрузки меняется на значок файла;
- установка больше не открывается сама автоматически;
- после скачивания нужно нажать на значок файла или блок загрузки, и уже откроется стандартное Android-окно установки/обновления приложения;
- если APK уже был скачан раньше, приложение не скачивает его заново, а сразу показывает кнопку/значок файла для установки.

Известные баги:
- на некоторых прошивках Android может не отдавать общий размер файла. Тогда будет показываться «размер неизвестен», но прогресс скачанных байт всё равно будет обновляться.
