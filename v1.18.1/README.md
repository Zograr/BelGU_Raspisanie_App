# v1.18.1

Переименование/хотфикс версии:
- версия переименована с v1.18 на v1.18.1;
- поднят `versionCode`, чтобы Android не считал установку откатом версии.

Содержит изменения v1.18:
- улучшенное меню настроек;
- карточка текущих параметров;
- быстрый доступ к оценкам, преподавателям, темам и обновлению;
- кнопка «Сбросить» в разделе оценок;
- сброс выбранного семестра, недели и последнего списка оценок.

Известные баги:
- функция оценок зависит от HTML таблицы dekanat.bsuedu.ru. Если сайт изменит разметку, понадобится хотфикс.

- versionName = 1.18.1
