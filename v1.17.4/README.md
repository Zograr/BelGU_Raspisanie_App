# v1.17.4

Хотфикс входа в журнал СПО:
- по видео было видно, что после входа сайт/страница журнала пытается открыться через `http://dekanat.bsuedu.ru/...`;
- добавлен перехват таких переходов внутри WebView;
- если WebView видит `http://dekanat.bsuedu.ru`, приложение автоматически пробует открыть `https://dekanat.bsuedu.ru`;
- ошибка `ERR_CLEARTEXT_NOT_PERMITTED` теперь не должна оставаться как финальная ошибка;
- добавлены более понятные сообщения при ошибке загрузки журнала.

Сохранено:
- установка APK через свой `ApkFileProvider`;
- HTTP/Cleartext config из v1.17.2;
- раздел «Оценки и посещаемость».

Известные баги:
- функция оценок всё ещё тестовая. После успешного входа может понадобиться доработка парсера таблицы.
