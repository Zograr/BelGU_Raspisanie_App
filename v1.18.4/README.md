# v1.18.4

Исправления журнала СПО:
- журнал грузится скрыто, WebView не должен мешать экрану;
- если нажать «Семестр» или «Неделя» слишком рано, приложение просит подождать «Журнал готов»;
- больше не должен открываться выбор языка RU/ENG;
- выбор недели больше не берёт список семестров 1-8;
- неделя ищется только по списку с датами/диапазонами недель;
- семестр и неделя больше не используют fallback на первый/последний select сайта.

Исправления настроек:
- убраны дубли кнопок, которые уже есть в быстром доступе;
- быстрый доступ стал основным местом для кнопок:
  - Оценки;
  - Преподы;
  - Темы;
  - Обновить;
- оставлены только настройки расписания и автообновления;
- дизайн быстрых кнопок сделан аккуратнее.

Технические фиксы:
- добавлены недостающие методы для «Сбросить выбор» и «Показать 2 / Н / НН»;
- убраны дубликаты методов;
- сохранена UTF-8 кодировка.
